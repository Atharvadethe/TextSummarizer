# Article Summarizer

A full-stack application that summarizes articles from URLs using advanced NLP models.

## Features

- Text preprocessing with NLTK
- Modern React frontend with HTML, CSS, Js
- Backend in Python Flask
- TF-IDF analysis
- Exteactive Summarization

  
## Prerequisites

- Python 3.8+
- Node.js 14+
- npm or yarn
